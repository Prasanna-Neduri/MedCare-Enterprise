# ✚ MedCare Pro | Enterprise Clinical Suite 2026

MedCare Pro is a high-end, dark-themed Medical Management System designed for seamless synchronization between patients and healthcare specialists. This application provides a unified interface for booking consultations, managing patient queues, and handling clinical feedback.

## ✨ Key Features

### 👤 For Patients (Students)
* **Specialist Discovery:** Filter doctors based on their assigned medical facility.
* **Smart Booking:** Reserve appointments with a specific **Reason for Visit** to prepare the doctor in advance.
* **Flexible Payments:** Choose between **Insurance**, **Credit Card**, or **Cash (On Visit)**.
* **Transparency:** Access a global feedback system to read and write reviews for specialists.

### 🩺 For Doctors (Medical Specialists)
* **Global Registry:** Dynamic registration that automatically adds the doctor to the hospital's public directory.
* **Patient Queue:** A real-time dashboard showing upcoming appointments, including the patient's specific symptoms/reasons.
* **Privacy-Focused Feedback:** A private review feed where doctors can see only the feedback directed at them.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid)
* **Typography:** Plus Jakarta Sans (Google Fonts)
* **Logic:** Vanilla JavaScript (ES6+)
* **Storage:** Browser LocalStorage (for persistent data simulation without a backend)
